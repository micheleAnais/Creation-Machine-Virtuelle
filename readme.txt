Dans la partie 1, j'ai installé VMware sur Windows et, sur VMware, j'ai installé une machine virtuelle ESXi. 
Ensuite, je suis allée sur la plateforme du serveur VMware en tapant `https://192.168.127.128/` dans un navigateur.
 Sur cette plateforme, je suis allée dans "Virtual Machines" pour créer deux machines virtuelles : 
TinyVM1 et TinyVM2. Pour créer le répertoire, je suis allée dans "Storage".
Dans la partie 2, il était question dans un premier temps de déployer notre fichier `.ova` sur une ou plusieurs
 machines virtuelles, facilitant la création automatique des machines virtuelles via un code disposant dudit 
fichier OVA, sans le faire manuellement comme nous l'avons fait pour les deux machines virtuelles TinyVM1 et 
TinyVM2. Ensuite, une fois notre machine virtuelle disposant du fichier OVA, il était question de la cloner 
pour obtenir une ou plusieurs machines virtuelles disposant également du fichier OVA. Enfin, il s'agissait de
 créer des machines virtuelles "from scratch".

J'ai pu réaliser la partie 1 ainsi que le déploiement du fichier OVA.
